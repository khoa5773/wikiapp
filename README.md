## Run

```bash
flutter pub get
flutter pub run flutter_launcher_icons:main
```

- Run with debug mode

```bash
flutter run
```

- Run with release mode

```bash
flutter run --release
```

## Release

- IOS

```bash
flutter build ios
```

- Android

```bash
flutter build apk
```
